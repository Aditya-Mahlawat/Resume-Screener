TO RUN THE APP

Terminal 1: Start the Backend

Navigate to the backend folder:
cd path/to/resume-screener/backend

Run the server:
uvicorn main:app --reload

Terminal 2: Start the Frontend

Navigate to the frontend folder:
cd path/to/resume-screener/frontend

Run the server:
npm run dev

After both are running, open http://localhost:3000 in your browser.

TO CLOSE THE APP

Go to Terminal 1 (where the backend is running).

Press Ctrl+C on your keyboard.

Go to Terminal 2 (where the frontend is running).

Press Ctrl+C on your keyboard.

Both servers are now stopped.
